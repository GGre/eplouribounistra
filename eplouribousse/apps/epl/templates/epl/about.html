***eplouribousse est une application web destinée à faciliter la gestion du dédoublonnement des périodiques dans les bibliothèques.***

Développée avec Django 2.2.12 (LTS) et Python 3.5.2, ReportLab open-source [3.5.42], HTML5, CSS3. Code en accès libre sur le GitLab de l'Université de Strasbourg ou sur GitHub.
